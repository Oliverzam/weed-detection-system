# Weed Detection and Quantification System using Mask R-CNN

Semantic segmentation web application for weed detection and quantification in potato crops using aerial drone imagery and Mask R-CNN.

## Project Description

This project focuses on the implementation of a deep learning-based semantic segmentation system for automatic weed detection in potato fields.

The system uses aerial images captured by drones and processes them using a Mask R-CNN architecture to identify and quantify weeds present in agricultural crops.

The solution follows a client-server architecture composed of:

- Frontend application
- Backend API
- Artificial Intelligence training module

# Dataset

The dataset was manually annotated using RoboFlow and consists of aerial drone images of potato crops containing weed instances.

---

# AI Model

The project uses the Mask R-CNN architecture for instance segmentation.

- Prediction visualization
